### 获取数据的API接口、字段等

#### 唯爱期货期权保证金

**接口**: `option_margin`

**目标地址**: https://www.iweiai.com/qiquan/yuanyou

**描述**: 唯爱期货-期权保证金

**限量**: 单次返回指定 symbol 的所有数据

**输入参数**

| 名称     | 类型  | 描述                                                    |
|--------|-----|-------------------------------------------------------|
| symbol | str | symbol="原油"; 可以通过 ak.option_margin_symbol() 所有 symbol |

**输出参数**

| 名称        | 类型      | 描述 |
|-----------|---------|----|
| 合约标的      | object  | -  |
| 合约代码      | object  | -  |
| 结算价       | float64 | -  |
| 交易乘数      | int64   | -  |
| 买方权利金     | int64   | -  |
| 卖方保证金     | float64 | -  |
| 手续费单位     | object  | -  |
| 开仓手续费     | int64   | -  |
| 平今手续费     | int64   | -  |
| 平昨手续费     | int64   | -  |
| 手续费(开+平今) | int64   | -  |
| 更新时间      | object  | -  |
