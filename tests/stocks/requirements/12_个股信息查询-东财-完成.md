### 获取数据的API接口、字段等

#### 个股信息查询-东财

**接口**: `stock_individual_info_em`

**目标地址**: http://quote.eastmoney.com/concept/sh603777.html?from=classic

**描述**: 东方财富-个股-股票信息

**限量**: 单次返回指定 symbol 的个股信息

**输入参数**

| 名称 | 类型 | 描述 |
|-----|-----|-----|
| symbol | str | symbol="603777"; 股票代码 |
| timeout | float | timeout=None; 默认不设置超时参数 |

**输出参数**

| 名称 | 类型 | 描述 |
|-----|-----|-----|
| item | object | - |
| value | object | - |
