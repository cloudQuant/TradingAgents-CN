### 获取数据的API接口、字段等

#### 财报发行

**接口**: `news_report_time_baidu`

**目标地址**: https://gushitong.baidu.com/calendar

**描述**: 百度股市通-财报发行

**限量**: 单次获取指定 date 的财报发行, 提供港股的财报发行数据

**输入参数**

| 名称 | 类型 | 描述 |
|-----|-----|-----|
| date | str | date="20241107" |

**输出参数**

| 名称 | 类型 | 描述 |
|-----|-----|-----|
| 股票代码 | object | - |
| 交易所 | object | - |
| 股票简称 | object | - |
| 财报期 | object | - |
