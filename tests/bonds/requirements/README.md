## 文档模板说明

每个需求文档包含以下部分：

1. **背景** - 说明为什么需要这个数据集合
2. **任务** - 明确需要实现的功能
3. **步骤** - 实现的详细步骤
4. **测试驱动** - 测试用例的要求
5. **验收标准** - 功能完成的标准
6. **获取数据的API接口、字段等** - AKShare接口详细说明
7. **实现要点** - 后端、前端、数据模型等实现细节
8. **扩展功能（可选）** - 可以进一步扩展的功能
