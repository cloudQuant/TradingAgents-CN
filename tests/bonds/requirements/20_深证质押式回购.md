#### 深证质押式回购

**接口**: `bond_sz_buy_back_em`

**目标地址**: https://quote.eastmoney.com/center/gridlist.html#bond_sz_buyback

**限量**: 单次返回所有行情数据

**输入参数**

| 名称 | 类型 | 描述 |
|----|----|----|

| 序号  | int64   | -  |
| 代码  | object  | -  |

| 涨跌额 | float64 | -  |
| 涨跌幅 | float64 | -  |
