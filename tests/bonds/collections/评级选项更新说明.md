## 📊 评级分类

### 投资级（Investment Grade）
- **AAA级**：AAA+, AAA, AAA-（极高信用质量）
- **AA级**：AA+, AA, AA-（高信用质量）
- **A级**：A+, A, A-（较高信用质量）
- **BBB级**：BBB+, BBB, BBB-（中等信用质量）

### 投机级（Speculative Grade / Junk Bonds）
- **BB级**：BB+, BB, BB-
- **B级**：B+, B, B-
- **CCC级**：CCC+, CCC, CCC-
- **CC级**：CC
- **C级**：C
- **D级**：D（违约）

### 短期评级
- **A-1**：最高短期评级
- **A-2**：高短期评级
- **A-3**：良好短期评级

## 💡 使用建议

### 查询高评级债券
```
评级: AAA 或 AAA+ 或 AA+
债券类型: 企业债
```

### 查询投资级债券
```
评级: AAA ~ BBB-（可多次查询）
```

### 查询短期融资券
```
债券类型: 短期融资券
评级: A-1 或 A-2
```

## ⚠️ 注意事项

1. **评级排序**
   - 下拉框按从高到低排序
   - AAA+ 最高，D 最低

2. **评级含义**
   - 投资级：BBB- 及以上
   - 投机级：BB+ 及以下
   - 短期评级：A-1、A-2、A-3

3. **查询建议**
   - 保守投资：选择 AA 及以上
   - 平衡投资：选择 A 到 AAA
   - 高收益：可选择 BB 及以下
