# 基金基本信息功能 - 快速使用指南

## 🚀 快速开始

### 1. 访问功能
1. 登录系统
2. 点击左侧菜单 **"基金投研"**
3. 点击子菜单 **"数据集合"**
4. 看到集合列表，找到 **"基金基本信息"** 卡片
5. 点击卡片进入详情页

**路径**: `/funds/collections/fund_name_em`

---

## 📊 首次使用

### 第一次更新数据

如果集合为空，需要先更新数据：

1. 点击右上角 **"更新数据"** 按钮
2. 在弹出对话框中查看更新说明
3. 点击 **"开始更新"** 按钮
4. 等待进度条完成（约10-30秒）
5. 看到成功提示：`成功更新 10,xxx 条数据`
6. 数据自动刷新到表格

**数据来源**: 东方财富网 (fund.eastmoney.com)

---

## 🔍 查看数据

### 数据表格

表格显示以下字段：
- **基金代码** - 6位数字代码
- **拼音缩写** - 基金名称拼音缩写
- **基金简称** - 基金完整名称
- **基金类型** - 混合型、股票型、债券型等
- **拼音全称** - 基金名称完整拼音

### 分页浏览

- **默认**: 每页50条
- **可选**: 20/50/100/200条
- **翻页**: 点击页码或使用上一页/下一页按钮
- **跳转**: 在输入框输入页码快速跳转

## 🔎 搜索和过滤

### 按字段过滤

**示例1**: 查找特定基金
```

```

**示例2**: 查找基金类型
```
过滤字段: 基金类型
过滤值: 混合型
点击 [搜索]
```

**示例3**: 按名称搜索
```
过滤字段: 基金简称
过滤值: 华夏
点击 [搜索]
```

### 清除过滤
点击输入框右侧的 **"×"** 清除，然后点击 **"搜索"**

---

## 📈 排序功能

### 按字段排序

1. 在"排序"下拉框选择字段（如：基金代码）
2. 选择排序方向：
   - **降序** - 从大到小 (Z→A, 9→0)
   - **升序** - 从小到大 (A→Z, 0→9)

**示例**:
```
排序字段: 基金代码
排序方向: 升序
→ 结果: 000001, 000002, 000003...
```

## 🔄 数据管理

### 刷新数据
- **功能**: 重新加载当前页数据
- **按钮**: 右上角 **"刷新"** 按钮
- **场景**: 查看最新数据状态

### 更新数据
- **功能**: 从东方财富网重新获取所有基金数据
- **按钮**: 右上角 **"更新数据"** 按钮
- **时间**: 约10-30秒
- **数据量**: 10,000+ 条记录

**更新流程**:
```
1. 点击"更新数据"
   ↓
2. 显示更新对话框
   ↓
3. 点击"开始更新"
   ↓
4. 显示进度条 (0% → 100%)
   ↓
5. 提示"成功更新 xxx 条数据"
   ↓
6. 自动刷新表格
```

### 清空数据
- **功能**: 清空集合中的所有数据
- **按钮**: 右上角 **"清空数据"** 按钮（红色）
- **警告**: ⚠️ 此操作不可恢复！

**清空流程**:
```
1. 点击"清空数据"
   ↓
2. 确认对话框
   "确认要清空...此操作不可恢复！"
   ↓
3. 点击"确认清空"
   ↓
4. 提示"成功清空 xxx 条数据"
   ↓
5. 表格显示无数据
```

## ⚠️ 注意事项

### 更新数据时
1. ⏱️ 更新需要时间，请耐心等待
2. 🌐 确保网络连接稳定
3. 🚫 更新期间不要关闭浏览器
4. ⚡ 建议在非高峰时段更新

### 清空数据时
1. ⚠️ 操作不可恢复
2. 💾 清空前确认不需要现有数据
3. 🔄 清空后需重新更新才有数据

### 过滤搜索时
1. 🔤 过滤值区分大小写
2. 🔍 支持模糊搜索
3. 🔄 记得点击"搜索"按钮

## ❓ 常见问题

### Q1: 为什么没有数据？
**A**: 首次使用需要点击"更新数据"从东方财富网获取。

### Q2: 更新需要多长时间？
**A**: 通常10-30秒，取决于网络速度和数据量。

### Q3: 数据多久更新一次？
**A**: 按需更新，建议每周或每月更新一次。

### Q4: 可以导出数据吗？
**A**: 当前版本暂不支持导出，后续版本会添加。

### Q5: 清空后数据能恢复吗？
**A**: 不能。清空后需重新点击"更新数据"获取。

### Q6: 过滤不到数据？
**A**: 检查：
- 过滤字段名是否正确
- 过滤值是否存在
- 是否点击了"搜索"按钮

### Q7: 更新失败怎么办？
**A**: 检查：
- 网络连接是否正常
- 查看错误提示信息
- 稍后重试
- 联系管理员
