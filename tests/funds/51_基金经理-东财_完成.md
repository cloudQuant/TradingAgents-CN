### 获取数据的API接口、字段等

基金经理
接口: fund_manager_em

目标地址: https://fund.eastmoney.com/manager/default.html

描述: 天天基金网-基金数据-基金经理大全

限量: 单次返回所有基金经理数据

输入参数

| 名称 | 类型 | 描述 |
|----|----|----|
| -  | -  | -  |

输出参数

| 名称        | 类型      | 描述       |
|-----------|---------|----------|
| 序号        | int64   | -        |
| 姓名        | object  | -        |
| 所属公司      | object  | -        |
| 现任基金代码    | object  | -        |
| 现任基金      | object  | -        |
| 累计从业时间    | int64   | 注意单位: 天  |
| 现任基金资产总规模 | float64 | 注意单位: 亿元 |
| 现任基金最佳回报  | float64 | 注意单位: %  |

接口示例

```python
import akshare as ak

fund_manager_em_df = ak.fund_manager_em()
print(fund_manager_em_df)
```
