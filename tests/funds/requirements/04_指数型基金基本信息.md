### 背景
在http://localhost:3000/funds/collections/fund_name_em这个数据集合里面，获取了基金的基本信息，但是并不完整，需要补充基金的详细信息。

### 任务

参考债券数据集合bond_info_cm的前端界面和后端实现的功能，从基金基本信息里面获取基金代码，参考下面获取数据的API接口和字段，获取信息，补充到雪球基金基本信息里面。

### 步骤
1. 数据集合：创建一个新的数据集合，名称为指数型基金基本信息
2. 页面里面需要包含数据概览、数据列表、刷新、清空数据、更新数据等功能、根据后端获取到的数据和数据列表里面的数据，增加一些基本的图形展示，让整个页面更加美观
3. 更新数据这个功能需要支持文件导入、远程同步、更新等功能。数据量不大，不需要批量更新了，一次更新全部的即可，这个数据集合以基金代码和日期作为唯一标识，更新数据的时候，如果存在重复的基金代码和日期，需要更新数据，否则需要插入数据。

### 获取数据的API接口、字段等
基金基本信息-指数型
接口: fund_info_index_em

目标地址: http://fund.eastmoney.com/trade/zs.html

描述: 东方财富网-天天基金网-基金数据-基金基本信息-指数型

限量: 单次返回当前时刻所有历史数据

输入参数

名称	类型	描述
symbol	str	symbol="全部"; choice of {"全部", "沪深指数", "行业主题", "大盘指数", "中盘指数", "小盘指数", "股票指数", "债券指数"}
indicator	str	indicator="全部"; choice of {"全部", "被动指数型", "增强指数型"}
输出参数

名称	类型	描述
基金代码	object	-
基金名称	object	-
单位净值	float64	-
日期	object	-
日增长率	float64	注意单位: %
近1周	float64	注意单位: %
近1月	float64	注意单位: %
近3月	float64	注意单位: %
近6月	float64	注意单位: %
近1年	float64	注意单位: %
近2年	float64	注意单位: %
近3年	float64	注意单位: %
今年来	float64	注意单位: %
成立来	float64	注意单位: %
手续费	float64	注意单位: %
起购金额	object	-
跟踪标的	object	-
跟踪方式	object	-
接口示例

import akshare as ak

fund_info_index_em_df = ak.fund_info_index_em(symbol="沪深指数", indicator="增强指数型")
print(fund_info_index_em_df)
