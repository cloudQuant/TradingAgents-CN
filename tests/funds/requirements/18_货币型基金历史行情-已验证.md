### 获取数据的API接口、字段等

货币型基金-历史数据
接口: fund_money_fund_info_em

目标地址: https://fundf10.eastmoney.com/jjjz_000009.html

描述: 东方财富网-天天基金网-基金数据-货币型基金-历史净值

限量: 单次返回当前时刻所有历史数据

输入参数

名称	类型	描述
symbol	str	symbol="000009"; 需要基金代码, 可以通过调用 ak.fund_money_fund_daily_em() 获取
输出参数

名称	类型	描述
净值日期	object	-
每万份收益	float64	-
7日年化收益率	float64	-
申购状态	object	-
赎回状态	object	-
