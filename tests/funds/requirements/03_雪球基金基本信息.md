### 获取数据的API接口、字段等

目标地址: https://danjuanfunds.com/funding/000001

描述: 雪球基金-基金详情

限量: 单次返回单只基金基本信息

输入参数

名称	类型	描述
symbol	str	symbol="000001"; 基金代码
timeout	float	timeout=None; 默认不设置超时参数
输出参数

名称	类型	描述
item	object	-
value	object	-
接口示例

import akshare as ak
