### 获取数据的API接口、字段等

净值估算-东财
接口: fund_value_estimation_em

目标地址: http://fund.eastmoney.com/fundguzhi.html

描述: 东方财富网-数据中心-净值估算

限量: 单次返回当前交易日指定 symbol 的所有数据

输入参数: symbol (基金类型): 全部, 股票型, 混合型, 债券型, 指数型, QDII, ETF联接, LOF, 场内交易基金

输出参数: 序号, 基金代码, 基金名称, 交易日-估算数据-估算值, 交易日-估算数据-估算增长率, 交易日-公布数据-单位净值, 交易日-公布数据-日增长率, 估算偏差, 交易日-单位净值

接口示例

```python
import akshare as ak

fund_value_estimation_em_df = ak.fund_value_estimation_em()
print(fund_value_estimation_em_df)
```
