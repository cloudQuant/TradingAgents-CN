### 获取数据的API接口、字段等

#### 供应维度

**接口**: `index_hog_spot_price`

**目标地址**: https://hqb.nxin.com/pigindex/index.shtml

**描述**: 行情宝-生猪市场价格指数

**限量**: 单次返回所有数据

**输入参数**

| 名称  | 类型  | 描述  |
|-----|-----|-----|
| -   | -   | -   |

**输出参数**

| 名称     | 类型      | 描述         |
|--------|---------|------------|
| 日期     | object  | -          |
| 指数     | float64 | -          |
| 4个月均线  | float64 | -          |
| 6个月均线  | float64 | -          |
| 12个月均线 | float64 | -          |
| 预售均价   | float64 | 注意单位: 元/公斤 |
| 成交均价   | float64 | 注意单位: 元/公斤 |
| 成交均重   | int64   | 注意单位: kg   |
